ANÁLISE DE DADOS DA EDUCAÇÃO PR

> date: 2024-05-20 > category: data_science > read_time: 5min

A educação pública gera terabytes de dados anualmente. O problema não é a falta de informação, mas a falta de processamento estratégico. Neste projeto, utilizei Python para analisar os microdados do estado e entender padrões ocultos.

// 01. O PROBLEMA

Como identificar escolas com desempenho discrepante em matemática na região metropolitana de Curitiba sem depender apenas de visitas presenciais?

WARNING: Dados anonimizados conforme LGPD. Nenhum aluno foi identificado neste estudo. Todos os dados utilizados são de domínio público.

// 02. A SOLUÇÃO TÉCNICA

Utilizei a biblioteca Pandas para limpeza de dados e Seaborn para visualização. O desafio inicial foi normalizar as planilhas do Saeb que vieram com cabeçalhos inconsistentes.

O código abaixo mostra como filtrei os dados regionais:

import pandas as pd
import seaborn as sns

# Carregando dataset
df = pd.read_csv('dados_pr_2023.csv')

# 1. Filtrando por região e disciplina
filtro = (df['region'] == 'metropolitana') & (df['materia'] == 'mat')

# 2. Agrupando por escola
resultado = df[filtro].groupby('escola')['nota_media'].mean()

# 3. Detectando anomalias (Outliers)
outliers = resultado[resultado < 4.0]
print(f"Escolas em alerta crítico: {len(outliers)}")

O resultado mostrou uma correlação direta entre a infraestrutura tecnológica disponível (labs de informática ativos) e o desempenho nas provas estaduais, sugerindo onde o investimento deve ser priorizado.

// 03. CONCLUSÃO

A análise de dados permite que a gestão escolar deixe de agir por "feeling" e passe a agir baseada em evidências sólidas. O próximo passo é aplicar um modelo de regressão linear para prever notas futuras.
